В данном репозитории представлен небольшое приложение платфрмы игр.

Приложение разработано с помощью:
Язык программирования - JavaScript;
Библиотека/Фреймворк - React;
State-менеджер - Storeon;
Оформление стилей - Styled-components.
Базовая конфигурация webpack для разработки.

Установка и запуск приложения
Установить и запустить приложение можно следующими действиями:

В вашей командной строке:

клонировать репозиторий
git clone https://github.com/rustam-ds/platform-games.git

перейти в папку проекта
cd platform-games

перейти в рабочую ветку
git checkout master

установка зависимостей
npm install

запуск приложения
npm start

Содержание приложения:
Входной файл приложения - index.js;
Стейт изолирован в папке store;
В папке components находятся компоненты, отвечающие за UI приложения;
В папке locales находится весь второстепенный текст приложения.
